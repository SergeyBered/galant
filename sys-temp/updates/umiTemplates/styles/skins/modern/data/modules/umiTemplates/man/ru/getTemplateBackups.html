Эта вкладка используется для управления резервными копиями (бэкапами) шаблонов ваших сайтов .

Общая информация

В списке отображены все резервные копии шаблонов ваших сайтов, отсортированные по дате создания (сначала самые новые).

Операции

На данной вкладке доступны четыре операции:

  • Для создания бэкапа нажмите на пиктограмму (Создать бэкап).
  • Для скачивания бэкапа нажмите на пиктограмму (Скачать бэкап).
  • Для восстановления шаблонов из бекапа нажмите на пиктограмму (Восстановиться из бэкапа).
  • Для удаление бэкапов нажмите на пиктограмму (Удалить бэкапы).
