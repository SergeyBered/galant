Эта вкладка используется для управления привязкой страниц к шаблонам.

Общая информация

На первом уровне дерева выводятся шаблоны, а на втором страницы. Поиск по имени работает только для страниц. Страницы без шаблона будут выведены в разделе "Без шаблона".

Привязка страниц

Изменить привязку страниц к шаблонам можно тремя способами:

  • Выбрать страницу(ы) и нажать на пиктограмму (Привязать другой шаблон для выбранных страниц).
  • Выбрать шаблон и нажать на пиктограмму (Привязать другой шаблон для всех страниц выбранного шаблона).
  • Выбрать страницу(ы) и перетащить в нужный шаблон.
