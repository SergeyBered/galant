На этой странице вы можете настроить параметры наложения водяного знака на изображения.

Водяной знак используется для защиты загружаемых на сайт изображений. Данная функция актуальна для фотографов и тех, кто публикует контент и не хочет, чтобы его использовали в коммерческих целях.

Для того, чтобы использовать водяные знаки необходимо в этой вкладке указать путь до заданного в качестве водяного знака изображения, указать прозрачность накладываемого изображения относительно исходника и его положение.

Чтобы водяной знак был наложен на изображения, необходимо загружать их через файловый менеджер с включенной опцией "Водяной знак".

В качестве изображения для водяного знака допустимо использовать изображения форматов: JPG, JPEG, PNG, WEBP, GIF.

Размер загружаемого изображения не должен превышать размер, указанный в модуле "Конфигурация" в поле "Максимальный размер загружаемой фотографии (Мб)".

Внимание: при первой установке или смене изображения, заданного в качестве водяного знака, необходима повторная загрузка всех защищаемых изображений.
