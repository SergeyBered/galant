Секция robots.txt предназначена для кастомизации файла robots.txt для доменов в системе.

Для кастомизации файла robots.txt нужно выбрать домен и нажать кнопку "Редактировать robots.txt" на тулбаре.

Для кастомизации можно использовать следующие макросы:

%disallow_umi_pages% - список страниц запрещенных для индексирования поисковыми системами

%host% - домен сайта

%crawl_delay% - время между запросами страниц для поисковых роботов заданное в config.ini секция [seo] параметр crawl-delay
