Вкладка Карта сайта предназначена для обновления файлов sitemap.xml (карта сайта) и sitemap-images.xml (карта изображений) для доменов в системе.

Эти карты задействованы при индексации сайтов в поисковых системах и улучшают SEO.

Для обновления карты сайта домена необходимо выбрать домен и нажать кнопку на тулбаре.

Для обновления карты изображений домена необходимо выбрать домен и нажать кнопку на тулбаре.

Карта изображений использует для своей работы данные карты сайта, поэтому для ее обновления необходимо предварительно обновить карту сайта.
