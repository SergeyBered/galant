Использовать торговые предложения в CommerceML —Позволяет импортировать торговые предложения со складами, остатками и типами цен из 1С.

Полная документация

Восстанавливать из модуля "Корзина" товары и разделы каталога при импорте из формата CommerceML — восстанавливает товары присутствующие в базе 1с, но перемещенные на сайте из каталога товаров в модуль "Корзина".

Изменять поле "h1" товаров и разделов каталога при импорте из формата CommerceML — изменяет поле "h1" при обновлении данных, если в 1С было изменено название номенклатуры ("ПолноеНаименование").

Изменять поле "title" товаров и разделов каталога при импорте из формата CommerceML — изменяет поле "title" при обновлении данных, если в 1С было изменено название номенклатуры ("ПолноеНаименование").

Записывать лог импорта в файл — Позволяет сохранять лог импорта в файл. Список файлов логов импорта можно посмотреть на вкладке "Логи импорта" в модуле Обмен данными.

Обновлять структуру каталога при импорте — Позволяет изменять структуру каталога при импорте, если она была изменена в источнике импорта (1С).
